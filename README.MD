# Ecommerce service
This mvc is responsible for:
* Onboarding users
* Roles and permissions
* Authentication
* email
* address

## Tech stack
* Build tool: maven >= 3.9.9
* Java: 21
* Framework: Spring boot 3.3.7
* DBMS: MySQL
* Kafka, zookeeper
* jwt, oauth2
* ...

## Prerequisites
* Java SDK 21
* A MySQL server
